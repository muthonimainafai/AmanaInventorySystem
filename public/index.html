Feed Inventory

Same day: a second save for the same brand, feed, and bag size adds to that day’s row. New day: one new row is created; quantity = yesterday’s remaining stock + today’s intake, and accumulated bags = previous total + today’s intake (example: 10 left + 20 new → quantity 30, accumulated 30). Use Clear before Save Record if the button still says “Update Record”.

Accumulated profit from today’s sales

Real-time running total for today’s feed sales (bags and kg). Example: margin KES 150, sell 1 bag -> 150; then 2 more bags -> 450.
